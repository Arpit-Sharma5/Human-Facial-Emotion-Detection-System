# Human Facial Emotion Detection System

This project detects human facial emotions from uploaded images using a trained deep learning CNN model.
It is built with Python, Flask, TensorFlow/Keras, and serves predictions through a simple web interface.
The model classifies emotions into seven categories: angry, disgust, fear, happy, neutral, sad, and surprise.
Users can upload an image and instantly view the predicted emotion with confidence score.
The application is deployed on Render for online access and real-time inference.
